# LBRY COTW

Simple script to generate an JSON list of all the Creators of the Week.

## Prerequisites
Requirements for running this
* [Bun](https://bun.sh/) (Supports Windows under WSL)

## Installation
```bash
bun install
```

## Run
```bash
bun start
```
